# Evidence Directory

Ten katalog przechowuje artefakty generowane podczas demo:

- `baseline.json` - obliczony baseline CC0
- `cc_timeseries.json` - seria czasowa CC
- `diagnosis_*.json` - diagnozy CFO
- `action_result_*.json` - wyniki Action Engine
- `so_execution_*.json` - logi Service Orchestrator

Po demo powinny istnieć wszystkie powyższe pliki.
